Autenticacao & Autorizacao

  • Proxy (proxy.ts): Intercepta todas as requisicoes, verifica sessao Supabase
  • Rotas publicas: /, /login, /cadastro, /recuperar-senha, /callback, /api/webhooks/*
  • Redirect: Nao autenticado? vai para /login?redirect=...
  • Redirect: Logado em pagina auth? vai para /dashboard
  • Admin: /admin/* requer role = 'admin' no profile (senao vai para /dashboard)
  • Hook client: useUser() para estado do usuario no client-side

Fluxo de Pagamento (Stripe)

  1. Usuario clica em plano na landing page ou area de membros
  2. POST /api/checkout cria sessao Stripe com price_id do plano
  3. Usuario completa pagamento no Stripe Checkout
  4. Stripe envia webhook checkout.session.completed
  5. Webhook cria/atualiza subscription no Supabase
  6. Auto-enroll: cria enrollment em todos os cursos publicados
  7. Em caso de cancelamento/falha: desativa enrollments automaticamente

▶ Proximos Passos

Fase A — Servicos Externos (Bloqueadores)

Estas tarefas precisam de acao manual para criar contas e obter chaves.

1

Configurar Stripe (Modo Teste)

1. Acessar dashboard.stripe.com (criar conta se nao tiver)
2. Ir em Developers → API keys e copiar pk_test_... e sk_test_...
3. Criar 3 Products com precos mensais recorrentes:
   • Iniciante: R$ 97,00/mes → copiar o price_...
   • Completo: R$ 197,00/mes → copiar o price_...
   • Presencial + Online: R$ 297,00/mes → copiar o price_...
4. Ir em Developers → Webhooks, criar endpoint:
   • URL: https://zelobjj.com.br/api/webhooks/stripe
   • Eventos: checkout.session.completed, customer.subscription.updated, customer.subscription.deleted, invoice.payment_failed
   • Copiar o whsec_...
5. Colar tudo no .env.local
6. Atualizar stripe_price_id nos planos do Supabase

2

Configurar Bunny Stream

1. Acessar dash.bunny.net (criar conta gratis)
2. Ir em Stream → Add Video Library
3. Copiar API Key, Library ID e CDN Hostname
4. Colar no .env.local
5. Fazer upload de pelo menos 1 video de teste

3

Configurar Resend

1. Acessar resend.com (criar conta gratis)
2. Ir em Domains e adicionar zelobjj.com.br
3. Adicionar registros DNS (DKIM, SPF) na Hostinger/Vercel
4. Ir em API Keys e criar uma chave re_...
5. Colar no .env.local

4

Reconectar Vercel + GitHub

1. Acessar Vercel → Settings → Git
2. Conectar o repositorio WendrewDouglas/ZeloBJJ branch master
3. Adicionar todas as env vars (Stripe, Bunny, Resend) em Settings → Environment Variables
4. Fazer um push para triggerar o deploy automatico
